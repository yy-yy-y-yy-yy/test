- Hyper Text Makeup Language
- [x] Hyper Text Markup Language
- Home Tool Markup Language

### Q
tags: IT, プログラミング
JavaScriptの変数宣言に使われないキーワードはどれですか？
- var
- let
- const
- [x] def
A: `def`はPythonやRubyなどで関数を定義する際に使われます。JavaScriptの変数宣言は`var`, `let`, `const`のいずれかです。
